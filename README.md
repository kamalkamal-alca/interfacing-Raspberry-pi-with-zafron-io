Zafron is a rudimentary and simple, but powerful dashboard, data logger, device registry and rules engine for IoT devices.     
Features
MQTT API
Device Registry
Device Logs
Rules & Alerts
Cloud to Device Commands
LoRaWAN support
HTTP API
Ingest data via HTTP
Learn More
To learn more about using Zafron with ESP devices, visit the ESP Guide.
For information on using Zafron with Arduino, see the Arduino Guide.
To understand how to migrate Cayenne projects to Zafron, refer to the Cayenne Projects Guide.
For detailed instructions on using Zafron with LoRaWAN, see the LoRaWAN Guide.
What do I need?
You can get started quickly with an existing device (Arduino or LoRa) or simulate data with a tool like mosquitto_pub or MQTT.fx.
Hardware An Arduino or ESP family board or any device connected to the Internet. The list of hardware that works with Zafron will keep growing.
Browser Zafron is designed to work on web browsers.
